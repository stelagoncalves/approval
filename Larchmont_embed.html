Zoning Board of Appeals

Download ZBA Application

Use when seeking variances to local zoning code, such as setbacks, height limits, or lot coverage.

Signatures Required:

  • ✔️ Applicant (notarized)
  • ✔️ Owner (notarized)

SWEC / SWPP Application

Download SWEC / SWPP Form

This permit is required for any construction activity involving:

  • Land disturbance ≥ 500 sq ft
  • Land disturbance < 500 sq ft as part of a larger common plan

Signatures Required:

  • ✔️ Applicant (Owner/ Contractor/ Architect) (non notorized)

Complete Application Checklist

  • □ Permit Application
  • □ Plans and Specifications (Two (2) Hard Copies) - FOLDED / NOT ROLL
  • □ Contractor's License & Insurance (Larchmont listed as Additional Insured)
  • □ Fees per Master Fee Schedule
  • □ Trade Permits (Plumbing, Electrical)
  • □ Special Inspections (if applicable)
  • □ Fire Code Compliance
  • □ Code Compliance Worksheet
  • □ Survey/Site Plan (IF EXTERIOR)
  • □ SWEC + Stormwater Plan (2 Hard Copies + 1 PDF) (IF EXTERIOR)
  • □ Zoning, Planning & ARB Approvals (IF EXTERIOR)

For more information, visit the official website: Larchmont Building Department
